# CharmApp

Progetto fullstack con PostgreSQL

## Struttura
- `/CharmApp-fe` - Frontend
- `/CharmApp-be` - Backend con PostgreSQL

## Installazione

### Backend
```bash
cd CharmApp-be
npm install
cp .env.example .env  # Configura le variabili d'ambiente
npm start
```

### Frontend
```bash
cd CharmApp-fe
npm install
npm start
```
